# Website screens

This directory is a navigation index for the website's deployable screens.

The HTML files intentionally remain at the repository root. Keeping `index.html` and the existing page filenames at their established paths allows the site to work on Vercel and ordinary static hosts without custom rewrite rules, and avoids rename conflicts when pages are edited on multiple branches.

## Main screens

- [`index.html`](../index.html) — home page
- [`About.html`](../About.html) — company information
- [`Blog.html`](../Blog.html) — blog index
- [`Contact.html`](../Contact.html) — contact page
